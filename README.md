Cette application permet l'affichage et la lecture des pages du Livre Sacré Le Coran.
Il est codé sous Delphi XE8.
Le code source est fourni dans sa totalité 
et sans aucune restriction sur les droits d'auteur.
Conseils:
1- Il faut respecter l'arborescence des répertoires
2- par contrainte sur l'espace disque autorisé, les fichiers audio et pages ne sont pas toutes disponibles
